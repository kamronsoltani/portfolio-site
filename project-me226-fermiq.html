FermIQ smart fermentation lid

2024 · ME226 · IoT · Food tech

FermIQ is a smart fermentation lid we built to make home and professional fermentation more consistent without expensive specialty hardware.

Background

FermIQ was built to make fermentation repeatable in both home and professional kitchens. It tracks the variables that actually matter, temperature, humidity, pH, and rise time, without forcing people into expensive closed hardware. We designed it as a modular system sized for standard containers so people can use what they already have.

Hardware

The lid uses a microprocessor based sensing stack to replace guesswork with continuous measurements and straightforward calibration.

Software

An ESP32 hosts a lightweight web server so users can monitor a ferment from a phone or laptop. Sensor data streams to the browser with live values for temperature, humidity, pH, and dough height. The interface includes a rise mode for proofing estimates and configurable thresholds so you can catch drift early instead of constantly doing manual checks.
