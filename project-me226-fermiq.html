FermIQ smart fermentation lid

2024 · ME226 · IoT · Food tech

A modular smart lid for home and pro kitchens, temperature, humidity, pH, and rise tracking over Wi-Fi so fermentation stays consistent from sourdough to kombucha.

Background

FermIQ was created to make fermentation easier to repeat at home and in professional kitchens. It targets the variables that actually drive outcomes, temperature, humidity, pH, and rise time, without the cost or lock-in of boutique appliances. The design is modular, priced for accessibility, and sized for standard containers so most cooks can adopt it.

Hardware

The physical story is told through CAD mockups and a sealed, sensor-rich lid: microprocessor-based sensing replaces guesswork with continuous measurements you can trust batch to batch.

Software

An ESP32 hosts a lightweight web server so you can monitor a ferment from a phone or laptop. Sensor data streams to the browser with live metrics for temperature, humidity, pH, and dough height. The interface includes a rise mode that estimates proofing time and configurable thresholds so you can intervene before a batch drifts off course, fewer manual spot checks without sacrificing control.
